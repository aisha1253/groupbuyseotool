QUICK FILL INSTRUCTIONS:
================================================================================

1. DATABASE_URL - Apna PostgreSQL connection string add karo:
   
   Local:
   DATABASE_URL="postgresql://postgres:mypassword@localhost:5432/groupbuytools"
   
   Supabase (Free):
   DATABASE_URL="postgresql://postgres:xxxxx@db.xxxxx.supabase.co:5432/postgres"
   
   Neon (Free):
   DATABASE_URL="postgresql://user:pass@ep-xxxxx.neon.tech/neondb?sslmode=require"

2. CLERK KEYS - Clerk Dashboard se:
   - https://dashboard.clerk.com par jao
   - API Keys section se Publishable Key aur Secret Key copy karo
   - Webhooks section se Webhook Secret copy karo

3. APP URL - Development me:
   NEXT_PUBLIC_APP_URL="http://localhost:3000"
